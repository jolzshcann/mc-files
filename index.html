Minecraft Modpack Installation
✔

1. Install Prism Launcher or CurseForge. (I recommend Prism)

⬇

2. Download the latest mod pack zip. DO NOT unzip it.

✔

3. Import the pack-latest.zip into either Prism Launcher or CurseForge.

✔

3a. (Optional) If using Prism, add the pack from direct download (insert link) to always get the latest version.

▶

4. Join the server and enjoy!
